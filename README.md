# 🔧 4라인 설비 알람 TOP5 대시보드

L2 알람 이력(4A/4B/4C/4X)을 분석하여 **정비 우선순위 TOP5**를 자동 산출합니다.

## 📊 데이터 스키마

| 컬럼 | 설명 | 예시 |
|---|---|---|
| 발생시간 | 알람 발생 | 2026-07-22 오후 10:39:52 |
| 해제시간 | 알람 해제 | 2026-07-22 오후 10:40:09 |
| 지속시간 | 지속시간 문자열 | 0시간 00분 16초 |
| 공정라인명 | 라인 구분 | 4라인 대입경 / 단결정 / 열처리 |
| 설비중분류 | 설비 구분 | 대입경A_본소성, 단결정C_1차해쇄 등 |

## 📐 TOP5 산정 로직

```
Score = w_freq × Norm(발생빈도) + w_dur × Norm(누적지속시간)
```
- 기본 가중치: 0.5 / 0.5 (UI에서 조정 가능)
- 정규화: Min-Max
- 집계 단위: **설비중분류**

## 🚀 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

`data/` 폴더에 `4A.csv`, `4B.csv`, `4C.csv`, `4X.csv`를 두거나 사이드바에서 업로드하세요.
